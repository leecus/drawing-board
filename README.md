# 🎨 画板 / drawing-board

一个运行在浏览器中的轻量画板应用，灵感来自 Windows 画板。

零依赖，打开即用。

## 功能

| 工具 | 快捷键 |
|---|---|
| 铅笔 (1px 硬边) | `P` |
| 画笔 (可调粗细) | `B` |
| 橡皮擦 | `E` |
| 直线 | `L` |
| 矩形 | `R` |
| 椭圆 | `C` |
| 油漆桶填充 | `G` |
| 文字 | `T` |

### 其他功能

- **撤销 / 重做** — `Ctrl+Z` / `Ctrl+Y`
- **保存为 PNG** — `Ctrl+S`
- **粗细调节** — 滑块 1-50px
- **28 色调色板** — 左键设前景色，右键设背景色
- **自定义颜色** — 取色器选择任意颜色
- **颜色互换** — 前景/背景色一键互换
- **清除画布** — 支持撤销恢复

## 使用

直接用浏览器打开 `index.html`：

```bash
open index.html
```

无需安装任何依赖，无需构建工具。

## 技术栈

- 纯 HTML5 + CSS3 + JavaScript (ES6)
- Canvas 2D API
- Pointer Events（同时支持鼠标和触控）
- 双 Canvas 分层（绘制层 + 预览层）
- 快照式 Undo/Redo
